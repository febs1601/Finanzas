Configurar Firebase
Solo hay que hacer esto una vez en cada celular. Toma 5 minutos.
Paso 1
Crear proyecto
Abre console.firebase.google.com → "Agregar proyecto" → ponle un nombre (ej: "felipe-vannia") → desactiva Analytics → "Crear proyecto"
Paso 2
Crear la base de datos
Menú izquierdo → "Realtime Database" → "Crear base de datos" → elige "Comenzar en modo de prueba" → "Habilitar"
Paso 3
Obtener las claves
⚙️ Configuración del proyecto → pestaña "General" → baja hasta "Tus apps" → botón "</>" (Web) → registra con cualquier nombre → copia los valores del objeto firebaseConfig
